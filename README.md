# 🌾 AI-Powered Sustainable Fertilizer Usage Optimizer for Sona Masuri Rice Yield Enhancement

An AI-powered fertilizer recommendation system that predicts optimal fertilizer dosage for Sona Masuri rice cultivation using soil nutrient values and machine learning models.

---

# 📌 Project Overview

This project helps farmers optimize fertilizer usage by analyzing:

- Soil Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Soil pH

The system predicts the required amount of:

- Urea
- SSP (Single Super Phosphate)
- MOP (Muriate of Potash)

using trained Machine Learning models.

---

# 🚀 Features

- Fertilizer recommendation using ML
- Random Forest & XGBoost models
- SHAP explainability for predictions

```bash

```

```bash

```

